# Пруд — эволюционный симулятор

Экспериментальный проект, демонстрирующий **эмерджентность** и **коэволюцию** в искусственной экосистеме.

## Что внутри

- 🟢 Водоросли — растут на свету, служат укрытием, выживают на дне
- 🐛 Дафнии — едят водоросли, эволюционируют (скрытность, тяга ко дну)
- 🐞 Жуки — едят дафний, не могут заходить на дно

## Три среды

| Среда | Рост водорослей | Охота жуков |
|-------|----------------|-------------|
| ☀️ Поверхность | Быстрый | Да |
| 💧 Открытая вода | Нормальный | Да |
| 🌿 Дно | Медленный | **Нет** |

## Эволюционные гены

- `stealth` — скрытность от жуков
- `bottom_feeding` — тяга к спуску на дно
- `speed`, `efficiency`, `neuroticism` — вспомогательные
